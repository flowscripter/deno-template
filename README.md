> Project template for a Deno executable.

## Usage

1. Export and expand a zip of this repository.
2. Update links and references in `README.md`.

## Development

Run: `deno run mod.ts`

Test: `deno test -A --unstable --coverage`

Lint: `deno fmt`

Compile: `deno compile --lite --unstable mod.ts`
